Movie API

This API provides a list of movies starring Jessica Chastain. Users can register, browse movies and see details about them, as well as make lists of their favorite movies.

Request URL HTTP method Request body data format Response body data format
Create a new user /users POST A JSON object with user data to be added, for example:
{
id: 1,
name: 'Susan',
favoriteMovies: ['Miss Sloane', 'The Tree of Life']
} 		A JSON object with data on student that has been added, including an id, for example:
{
id: 1,
name: 'Susan',
favoriteMovies: ['Miss Sloane', 'The Tree of Life']
id: 21511
}
Create new favorite movie for user /users/:id/:movieTitle POST None A confirmation message that the movie has been added, for example: 'Miss Sloane has been added to user 2 array.'
Return a list of all movies to the user /movies GET None An array of JSON objects, each one containing data about movies in the database.
Return details about a movie to the user /movies/:title GET None A JSON object containing details about the movie, for example:
{
title: 'A Most Violent Year',
director: 'J.C.Chandor',
genre: ['action', 'crime', 'drama']
}
Return genre of the movie by title /movies/genre/:title GET None An array containing genre descriptions of the movie, for example:
[
"drama"
]
Return data about a director /movies/director/:directorName GET None A JSON object containing info about a director, for example:
{
name: 'Aaron Sorkin',
year of birth: '1973',
country of birth: 'USA']
}
Update user details /users/:id PUT A JSON object containing the data to be updated in user details, for example:
{
name: 'Suzie'
}		 A JSON object containing updated user data, for example:
{
id: 1,
name: 'Suzie',
favoriteMovies: ['Miss Sloane', 'The Tree of Life']
}
Delete a favorite movie from a user's list /users/:id/:movieTitle DELETE None A message confirming that the data has been deleted from the list.
Delete (deregister) a user /users/:id DELETE None A message confirming that the user has been deleted from the registry.
