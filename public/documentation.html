Movie API

This API provides a list of movies starring Jessica Chastain. Users can register, browse movies and see details about them, as well as make lists of their favorite movies.

Request URL HTTP method Request body data format Response body data format
Create a new user /users POST A JSON object with user data to be added, for example:
{
Username: 'Susan',
Password: 'Sus2024',
Email: 'susan.lewis@gmail.com',
Birthday: '1991-05-23',
FavoriteMovies: ["6368faaadec8a0c53e3a6d3d"]
} 		A JSON object with data on user that has been added, including an id, for example:
{
id: 1bjdf,
Username: 'Susan',
Password: 'Sus2024',
Email: 'susan.lewis@gmail.com',
Birthday: '1991-05-23',
FavoriteMovies: ["6368faaadec8a0c53e3a6d3d"]
}
Create new favorite movie for user /users/:Username/movies/:MovieID POST None A JSON object with user's updated data, for example:
{
id: 1bjdf,
Username: 'Susan',
Password: 'Sus2024',
Email: 'susan.lewis@gmail.com',
Birthday: '1991-05-23',
FavoriteMovies: ["6368faaadec8a0c53e3a6d3d", "6368f765dec8a0c53e3a6d3a"]
}
Return a list of all movies to the user /movies GET None An array of JSON objects, each one containing data about movies in the database.
Return details about a movie to the user /movies/:title GET None A JSON object containing details about the movie, for example:
{
Title: 'A Most Violent Year',
Description: '...',
Released: '2014',
Genre: {Name: 'Crime', Description: '...'},
Director: {Name: 'J.C.Chandor', Bio: '...', Birth: '1974'},
Actors: ['Oscar Isaac', 'Jessica Chastain'],
ImagePath: 'amostviolentyear.jpg',
Featured: true}
Return genre details by genre name /movies/genre/:genreName GET None A JSON object containing details about the genre, for example:
{
Name: 'drama',
Description: 'In film and television, drama is a category or genre of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone.'
}
Return data about a director by name /movies/director/:directorName GET None A JSON object containing info about a director, for example:
{
Name: 'Aaron Sorkin',
Bio: '...',
Birth: '1973'
}
Update user details /users/:Username PUT A JSON object containing the data to be updated in user details, for example:
{
Name: 'Suzie'
}		 A JSON object containing updated user data, for example:
{
id: 1bjdf,
Username: 'Suzie',
Password: 'Sus2024',
Email: 'susan.lewis@gmail.com',
Birthday: '1991-05-23',
FavoriteMovies: ["6368faaadec8a0c53e3a6d3d"]
}
Delete a favorite movie from a user's list /users/:Username/movies/:MovieID DELETE None A JSON object with user's updated data, for example:
{
id: 1bjdf,
Username: 'Susan',
Password: 'Sus2024',
Email: 'susan.lewis@gmail.com',
Birthday: '1991-05-23',
FavoriteMovies: ["6368faaadec8a0c53e3a6d3d"]
}
Delete (deregister) a user /users/:Username DELETE None A message confirming that the user has been deleted from the registry.
